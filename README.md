# 智能超市收银系统 (Smart Supermarket POS System)

基于C++/Qt开发的现代化智能超市收银系统，集成条形码识别和AI推荐功能。

## 系统特性

### 核心功能
- **商品管理**: 完整的CRUD操作，支持条码、价格、库存管理
- **销售结账**: 流畅的收银流程，支持多种支付方式
- **库存跟踪**: 实时库存更新和低库存预警
- **票据打印**: 标准POS票据格式输出
- **用户权限**: 基于角色的访问控制

### 创新功能
- **实时条形码识别**: 基于QZXing库的摄像头扫描，替代传统扫描枪
- **AI商品推荐**: 混合推荐算法，基于用户购买历史进行智能推荐

## 技术架构

### 技术栈
- **开发语言**: C++17
- **UI框架**: Qt 6.x (Qt Widgets)
- **数据库**: SQLite
- **条形码识别**: QZXing
- **构建系统**: CMake

### 架构设计
```
表现层 (Qt Widgets)
    ↓
业务逻辑层 (Controllers)
    ↓
数据访问层 (DatabaseManager)
    ↓
数据存储 (SQLite)
```

## 快速开始

### 环境要求
- Qt 6.0 或更高版本
- CMake 3.16 或更高版本
- C++17 兼容编译器
- SQLite 3
- 支持的摄像头设备

### 构建步骤
```bash
mkdir build
cd build
cmake ..
make
```

### 运行系统
```bash
./SmartPOS
```
